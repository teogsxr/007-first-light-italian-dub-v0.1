# Pipeline source-first

1. Mapping WEM ufficiale e sottotitoli.
2. Casting voce target per personaggio.
3. Generazione italiana source-first: source audio come performance authority, target voice come identity authority.
4. ASR QA: testo parlato, required words, nessun tag letto.
5. Conformance: timing trimmed, energia, pause, emozione/performance.
6. Filtro semantico/editoriale.
7. Mastering 48 kHz mono PCM16.
8. Wwise Vorbis encode.
9. Marker preservation.
10. Runtime safety/capacity gate.
11. Subtitle-link audit pre-apply e cumulativo.
12. Patch in-place solo dei WEM promossi.
