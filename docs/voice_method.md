# Metodo voci

Le voci target sono template/varianti approvate per personaggio. La resa finale non ? un clone diretto del take inglese: la voce target resta l'identit?, mentre l'audio sorgente guida performance, intenzione, pause, respiri e intensit?.

La variante senza patos, usata nei test locali, viene generata usando solo la voce target e il testo italiano. La variante source-first aggiunge il source audio come riferimento di performance.
